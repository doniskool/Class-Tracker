# Class-Tracker

A sleek grade tracking web app for recording classwork, homework, and average grades.

## Features

- Add grade entries with class, type, score, total, weight, and notes
- Track a current average grade alongside your baseline grade of 0%
- See counts for homework, classwork, and total entries
- Remove individual entries or reset the tracker
- Smooth bacterium-inspired background and button click sound effects
- Background music player with playlist (Lofi, Let's larp, Romantical and more to come).
- Custom background image uploader
- Customer support contact - send me music recommendations
- Completely client side, multiple people can use the website at the same time.


## Usage

Open `index.html` in a browser to use the tracker, or visit the live site at https://doniskool.github.io/Class-Tracker/

Enter your grades manually and watch the average update automatically.

## Music Player

Click the spinning disk in the bottom-right to access the music player. Select a song, play/pause, and adjust volume.
